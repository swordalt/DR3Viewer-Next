# DR3Viewer-Next

DR3Viewer-Next is an updated fork of [DanceRail3Viewer](https://github.com/lucarioex/DanceRail3Viewer) by [lucarioex](https://github.com/lucarioex). It remains true to the original project and hence the official game.

## Usage
Usage is the same as the original project.
1. Download Unity version 2021.2.17f1. Open the project.
2. Move/copy your relevant audio/chart/illustration files into their respective folders within `Assets/Resources`.
4. Click on `game.unity` to load the game scene.
5. Change the keyword and difficulty (required), as well as any other field in `The Game Manager`.
6. Press the play button. Playback will start automatically.

**NOTE**: This project was designed for use within the Unity editor. Building is not recommended.<br>

## Changes
- Added support (and sprites) for note types 25, 26, and 27.
- Fixed an issue when parsing note parent data.
- Fixed an animation warning when a SC occurs.
